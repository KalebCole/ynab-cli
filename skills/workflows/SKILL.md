# Workflows Skills

Complex multi-step procedures and common YNAB workflows.

## Available Skills

- **expense-tracking**: Track and categorize expenses  
- **budget-setup**: Initial budget configuration
- **reconciliation**: Account reconciliation procedures

## Purpose

Workflows combine multiple CLI operations into cohesive procedures that accomplish common YNAB tasks. Each workflow provides step-by-step guidance for complex operations.

## Types

- **Setup Workflows**: Initial configuration and onboarding
- **Maintenance Workflows**: Regular budget maintenance tasks  
- **Analysis Workflows**: Data analysis and reporting procedures
- **Troubleshooting Workflows**: Problem resolution and cleanup

## Benefits

- Standardized procedures for common tasks
- Reduced learning curve for new users
- Consistent execution of complex operations
- Integration with automation systems

## Related Skills

Workflows orchestrate other skills from:
- Transactions (data entry and management)
- Categories (budget allocation)  
- Accounts (account management)
- Reports (analysis and insights)
