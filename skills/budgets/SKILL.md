# Budgets Skills

Budget management and selection operations.

## Available Skills

- **list**: List all accessible budgets
- **set-default**: Set default budget for CLI operations

## Core Capabilities

- Budget discovery and listing
- Default budget configuration  
- Budget details and metadata access
- Settings management

## Key Features

- Support for multiple budgets per account
- Default budget configuration for seamless CLI usage
- Budget metadata including last modification times
- Settings access for currency and date preferences

## Related Skills

- Auth (required for budget access)
- All data operations (use budget context)
- Accounts, Categories, Transactions (budget-scoped)
