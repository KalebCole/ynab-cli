# YNAB CLI Skills

This directory contains skill definitions for AI agents working with the YNAB CLI. Each SKILL.md file describes specific capabilities and provides structured guidance for automated interactions.

## Directory Structure

- **auth/**: Authentication and authorization operations
- **budgets/**: Budget management and selection 
- **accounts/**: Account operations and queries
- **transactions/**: Transaction management (CRUD, search, analysis)
- **categories/**: Category operations and budget allocation
- **payees/**: Payee management and operations
- **reports/**: Data analysis and reporting capabilities
- **workflows/**: Complex multi-step procedures

## Using Skills

Each SKILL.md file follows a standard format:
- **Description**: What the skill does
- **Usage**: Command syntax and examples  
- **Parameters**: Required and optional parameters
- **Output**: Expected response format
- **Use Cases**: Common scenarios and workflows
- **Related Skills**: Connected operations

## Skill Categories

### Core Operations
- Authentication (login/logout)
- Data retrieval (list operations)
- Basic CRUD (create/update/delete)

### Advanced Features  
- Transaction splitting and analysis
- Transfer matching and reconciliation
- Budget analysis and reporting

### Workflows
- Expense tracking procedures
- Budget setup and management
- Account reconciliation processes
